How it works

This module wraps InsightFace with a pluggable storage backend, exposing four core operations through a clean Python API.

1

Enroll

Upload a single-face photo to register a person. The module extracts a 512-d embedding and stores it.

2

Identify (1:N)

Upload any photo. Every face is detected and compared against all enrolled embeddings.

3

Verify (1:1)

Upload a photo and pick a person — the module tells you if it's a match with a confidence score.

4

Manage

List all enrolled people, inspect metadata, or remove entries from the database.

👤 Enroll a Person

Upload a photo with exactly one face. The module extracts a 512-dimensional face embedding and stores it with the given ID and name.

🔍 Identify Faces (1:N)

Upload a photo containing one or more faces. The module detects every face, compares each against all enrolled embeddings, and returns the best match (or "Unknown" if below the threshold). The output image is annotated with bounding boxes.
